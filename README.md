# Simple library

# Functional
- Authorization and registration
- JWT token for permission
- User, Admin roles
- Delete, add books and authors

# Database
- sqlite

# Linters
- black
- isort
- ruff

# Installing requirements
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- uvicorn src.main:app --reload
